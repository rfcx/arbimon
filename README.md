# Biodiversity Analytics

Biodiversity analytics website build with Vue 3 + Typescript + Vite + Windi CSS

## Getting started

Requirements:
- node 14+, npm 7+

1. Install dependencies:

    `npm install`

2. Copy `.env.example` to `.env` and field in all values

3. Start developing an application:

    `npm run serve`

4. Build package:

    `npm run build`
