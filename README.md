# Biodiversity Analytics

Biodiversity analytics website build with Vue 3, Typescript, Vite, Windi CSS, and Vuex

## Requirements

## Getting started

1. Install dependencies:

    `npm install`

2. Start developing an application:

    `npm run serve`

3. Build package:

    `npm run build`
